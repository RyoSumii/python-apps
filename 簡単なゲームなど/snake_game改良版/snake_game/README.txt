[説明]
古典的なゲームである"Snake Game" またはヘビゲームです。
同一のディレクトリにすべてのpythonファイルがあれば実行できます。
main.py の SLEEP_TIME を変えることでヘビのスピードを変えることが出来ます。
またsnake.pyのMOVE_DISTANCEの数値を下げることで、疑似的にfpsをあげることが出来ます。

[改良版]
data.txtで最高スコアを記録します。

[操作方法]
キーボードのエンターキー下にある↑↓→←のキーでヘビの進行方向を変えることが出来ます。
